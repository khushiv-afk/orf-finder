# ORF Finder

## Overview

This project implements a Python script to identify Open Reading Frames (ORFs) in a DNA sequence. ORFs represent potential protein-coding regions defined by a start codon (ATG) and a stop codon (TAA, TAG, TGA).

## Features

* Detects multiple ORFs in a sequence
* Outputs ORF sequences along with their lengths

## Example

Input:
ATGAAATGAAAAATAGATGTAA

Output:
ATGAAATGA (9)
ATGAAAAATAG (12)
ATGTAA (6)
